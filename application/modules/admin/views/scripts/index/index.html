• 「御社名 or サイト」ショップカスタマイズ部
  • ・コンテンツ検索
  • 　ショップで扱っているコンテンツを検索します。またコンテンツごとにキーワードを検索できます。
  • ・ピックアップ枠コンテンツ編集
  • 　トップページなどピックアップ枠に表示するコンテンツを編集します。コンテンツの指定はBookIDで行います。
  • ・フリーテキスト編集
  • 　コンテンツにひもづかないテキストを編集します。サイト構築時にテンプレートにフリーテキスト枠を埋め込んでおく必要が
  • 　あります。フリーテキストは管理が大変なので、できるだけ使わないようにしましょう。
  • ・本番サーバーのショップDB・ファイル更新
  • 　本番サーバーのショップが検証サーバーと同じ内容に更新されます。
